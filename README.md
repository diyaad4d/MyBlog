## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/diyaad4d/MyBlog.git
cd MyBlog
```
2. Install dependencies:

```bash
npm install
```

3. Start the API server:

```bash
node server.js
```

4. Start the frontend server:

```bash
node app.js
```

5. Open your browser:

``` bash
http://localhost:3000
```
